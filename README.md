# SMS Spam Collector

This repository contains the academic project report for **SMS Spam Collector**, developed as part of my Bachelor of Computer Applications (BCA).

## Project Overview
SMS Spam Collector is a project focused on identifying spam and non-spam (ham) SMS messages.  
The project explains the use of text processing and basic machine learning concepts to classify messages.

## Objective
- To understand SMS spam detection
- To learn fundamentals of NLP and Machine Learning
- To apply theoretical concepts in a practical academic project

## Concepts Covered
- Text preprocessing
- Spam vs Ham classification
- Machine Learning basics
- Natural Language Processing (NLP)

## Technologies Mentioned
- Python
- Machine Learning
- NLP (Natural Language Processing)
